Tutorial dell'elemento Form

La tipografia web riguarda l'aspetto del testo sul tuo sito web. Non si tratta solo di scegliere un font, ma anche di gestire spaziature, dimensioni, allineamenti e l'armonia complessiva tra gli elementi tipografici.

La tipografia ha un impatto rilevante per la leggibilità, l'accessibilità e l'estetica del sito.

Cominciamo imparando la terminologia di base della tipografia:

Font

Approfondiamo la differenza tra font personalizzati e font di sistema.

Font di sistema

Si tratta di font preinstallati su dispositivi che garantiscono velocità e compatibilità.

Font web

Si tratta di font scaricati da Internet, per esempio da Google Fonts o Adobe Fonts, che offrono maggiore varietà e personalizzazione.

Un font web è un tipo di carattere tipografico progettato per essere utilizzato su pagine web. A differenza dei font tradizionali che sono installati sul tuo computer, i font web sono caricati direttamente da Internet quando visiti un sito.

Questo permette ai web designer di utilizzare una vasta gamma di caratteri personalizzati per rendere le pagine web più belle e leggibili, indipendentemente dai font che gli utenti hanno installato sui loro computer.

Così, quando visiti un sito, vedi esattamente i caratteri che il designer ha scelto, rendendo l'esperienza di lettura più uniforme e piacevole.

Di default, il browser con cui stai navigando applica delle proprie regole, e quindi un proprio font, al sito web; queste regole di base vengono poi sovrascritte dalle regole “personalizzate”, inserite dai fogli di stile (css).

Per creare una propria regola personalizzata relativa al font da utilizzare, la sua dimensione e tutte le altre proprietà, si può utilizzare la regola “@font-face” che, successivamente, verrà applicata al tag html (o all’elemento, tramite classe o ID) scelto.

Vediamo un esempio.

Download font from Google or Adobe file with extension .woff2,.ttf add to CSS style sheet this row

@font-face {

font-family: 'MyCustomFont';

src: url('../fonts/Edu_AU_VIC_WA_NT_Guides/EduAUVICWANTGuides-VariableFont_wght.ttf') format('truetype'); /* Safari, Android, iOS */

}
