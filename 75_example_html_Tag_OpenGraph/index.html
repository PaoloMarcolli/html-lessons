Tutorial tabelle

Fonte immagine

Le tabelle in HTML sono utilizzate per organizzare i dati in righe e colonne. Sono ideali per presentare informazioni tabellari, come orari, report finanziari, listini prezzi e molto altro.

In questo capitolo, esploreremo la creazione di tabelle con HTML e la loro formattazione con CSS per renderle visivamente accattivanti e facilmente leggibili.

Struttura base di una tabella HTML

Una tabella HTML è definita utilizzando l'elemento <table>

  • <theader>: contiene l'intestazione della tabella.
  • <tbody>: contiene il corpo della tabella, dove sono inseriti i dati..
  • <tr>: definisce una riga della tabella.
  • <th>: definisce una cella di intestazione della tabella.
  • <td>: definisce una cella di dati della tabella.

Vediamo un esempio di una tabella di base.

                    
                        html: 

                        <table>
                        <thead>
                            <tr>
                                <th>Nome</th>
                                <th>Età</th>
                                <th>Città</th>
                            </tr>
                        </thead>
                        <tbody>
                            <tr>
                                <td>Mario Rossi</td>
                                <td>30</td>
                                <td>Roma</td>
                            </tr>
                            <tr>
                                <td>Luigi Bianchi</td>
                                <td>25</td>
                                <td>Milano</td>
                            </tr>
                        </tbody>
                    </table>

Formattazione delle tabelle con CSS

Per rendere le tabelle più leggibili e visivamente accattivanti, possiamo utilizzare CSS.

Ecco alcune proprietà CSS comuni per formattare le tabelle:

  • border: aggiunge bordi alle celle della tabella.
  • padding: aggiunge spazio interno alle celle.
  • text-align: allinea il testo nelle celle.
  • background-color: cambia il colore di sfondo delle celle.
  • nth-child: selettore avanzato per applicare stili a righe specifiche.

Vediamo un esempio di formattazione CSS per una tabella.

                    
                        css:

                        table {
                          width: 100%;
                          border-collapse: collapse; /* Elimina lo spazio tra i bordi delle celle */
                        }
                        
                        th, td {
                          border: 1px solid #dddddd;
                          text-align: left;
                          padding: 8px;
                        }
                        tr:nth-child(even) {
                          background-color: #f2f2f2; /* Colore di sfondo per le righe pari */
                        }
                        th {
                          background-color: #4CAF50;
                          color: white;
                        }

Tabelle responsive

Le tabelle possono diventare difficili da leggere su dispositivi mobili.

Per rendere una tabella responsive, possiamo utilizzare CSS per farla scorrere orizzontalmente su schermi più piccoli.

Vediamo un esempio di tabella responsive.

                    
                        css:

                        table {
                          width: 100%;
                          border-collapse: collapse;
                        }
                        th, td {
                          border: 1px solid #dddddd;
                          text-align: left;
                          padding: 8px;
                        }
                        tr:nth-child(even) {
                          background-color: #f2f2f2;
                        }
                        th {
                          background-color: #4CAF50;
                          color: white;
                        }
                        @media screen and (max-width: 600px) {
                          table {
                            width: 100%;
                            display: block;
                            overflow-x: auto; /* Aggiunge uno scorrimento orizzontale */
                          }
                        }

Le tabelle responsive sono utilizzate per organizzare informazioni con una larga quantità di colonne e righe.

Tabelle complesse

Le tabelle possono contenere intestazioni di colonna e di riga, celle che attraversano più righe o colonne, e altro ancora.

Analizziamo un esempio di tabella con celle che attraversano più colonne e righe.

                    
                        html :

                        <table>
                        <thead>
                          <tr>
                            <th>Nome</th>
                            <th>Età</th>
                            <th>Città</th>
                            <th rowspan="2">Note</th> <!-- Cell che attraversa due righe -->
                          </tr>
                          <tr>
                            <th colspan="2">Informazioni Personali</th> <!-- Cell che attraversa due colonne -->
                            <th>Contatti</th>
                          </tr>
                        </thead>
                        <tbody>
                          <tr>
                            <td>Mario Rossi</td>
                            <td>30</td>
                            <td>Roma</td>
                            <td rowspan="2">Nessuna nota</td> <!-- Cell che attraversa due righe -->
                          </tr>
                          <tr>
                            <td>Luigi Bianchi</td>
                            <td>25</td>
                            <td>Milano</td>
                          </tr>
                        </tbody>
                      </table>
