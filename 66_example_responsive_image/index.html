Esempio di Immagini Resposive

Le immagini responsive sono un elemento essenziale per garantire una buona esperienza utente su dispositivi con dimensioni dello schermo e risoluzioni diverse

Questo capitolo del Corso ti guiderà attraverso i vari metodi per rendere le immagini del tuo sito web responsive.Parlando di design responsive, abbiamo imparato a utilizzare le media query per creare layout separati per dispositivi mobili, tablet e desktop. Ora vogliamo fare lo stesso con le immagini. Proprio come le media query ci permettono di presentare condizionatamente diverse regole CSS, vogliamo visualizzare diverse immagini in base al dispositivo dell'utente.

Le immagini hanno dimensioni intrinseche.

Non possiamo allungare una foto di 500×250 pixel oltre i 500 pixel di larghezza senza che diventi pixelata. I display retina e i dispositivi mobili complicano ulteriormente le cose. Per rendere le nostre immagini responsive, dobbiamo considerare tre aspetti:

  • Le dimensioni del dispositivo
  • Le dimensioni dell'immagine
  • La risoluzione dello schermo del dispositivo

Per sperimentare con le immagini responsive, abbiamo bisogno di un sito web responsive con cui lavorare. In questo capitolo, costruiremo una pagina web di esempio aggiungendo due immagini che cambieranno a seconda del dispositivo dell'utente.

Schermi retina

I display retina hanno il doppio dei pixel per pollice rispetto ai display a risoluzione standard.

Per rendere correttamente un'immagine su un dispositivo retina, l'immagine deve essere grande il doppio delle sue dimensioni di visualizzazione finali.Per esempio, se vuoi aggiungere un'immagine di 500×250 pixel alla pagina, il file dell'immagine corrispondente deve essere di 1000×500 pixel.
