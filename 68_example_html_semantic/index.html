Definire una grid

Per definire una grid bisogna bisogna valorizzare la property display : grid nella classe

Grid flessibili con l'unita fr

  • Per definire una grid flessibili bisogna bisogna valorizzare la property grid-template columns : 1fr 1fr 1fr nella classe

    Il numero massimo di colonne per pagina è 12 ovvero : 12 fr
