Bootstrapt


Bootstrap è uno dei framework front end più popolari, sviluppato da X (in precedenza Twitter). È un framework open source che facilita la creazione di siti web responsivi e mobile-first.

Bootstrap include una vasta gamma di componenti CSS e JavaScript predefiniti che possono essere utilizzati per costruire interfacce utente moderne.

Il concetto alla base è che includendo i sorgenti di Bootstrap (CSS, JS) nell’header della pagina, anche tramite CDN, è possibile, assegnando delle classi prestabilite ai nostri elementi HTML, renderli esteticamente accattivanti e funzionali.

Analizziamo più da vicino le caratteristiche di Bootstrap.

Sistema di griglia

Un sistema di griglia flessibile e responsivo che consente di creare layout complessi in modo semplice (ovviamente è basato su Flexbox, che hai già avuto modo di vedere nelle sezioni precedenti).

Il concetto alla base è quello di creare un “container” al cui interno è possibile inserire un numero infinito di righe, ognuna delle quali può essere suddivisa in un massimo di 12 colonne.

Ovviamente ogni colonna potrà essere dell’altezza e larghezza desiderata e potrà contenere, a sua volta, altre righe con altrettante colonne.

Vediamo un esempio.

        
        html:

        <div class="container">
        <div class="row">
          <div class="col-md-4">Colonna 1</div>
          <div class="col-md-4">Colonna 2</div>
          <div class="col-md-4">Colonna 3</div>
        </div>
      </div>

JavaScript plugin

I plugin JavaScript sono moduli o librerie che estendono le funzionalità di un sito web. Sono progettati per essere facilmente integrabili e riutilizzabili, permettendo agli sviluppatori di aggiungere nuove caratteristiche e comportamenti senza dover scrivere codice da zero.

Questi plugin possono gestire una varietà di compiti, come animazioni, modali, slider, tooltip e molto altro.

Bootstrap, all’interno della propria libreria JavaScript, offre tantissimi plugin integrati pronti all’uso per aggiungere funzionalità interattive, come modali, tooltip e popover.

Vediamo un esempio.

      
        html:

      <!-- Button trigger modal -->
      <button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exampleModal">
        Launch demo modal
      </button>
      
      <!-- Modal -->
      <div class="modal fade" id="exampleModal" tabindex="-1" aria-labelledby="exampleModalLabel" aria-hidden="true">
        <div class="modal-dialog">
          <div class="modal-content">
            <div class="modal-header">
              <h1 class="modal-title fs-5" id="exampleModalLabel">Modal title</h1>
              <button type="button" class="btn-close" data-bs-dismiss="modal" aria-label="Close"></button>
            </div>
            <div class="modal-body">
              ...
            </div>
            <div class="modal-footer">
              <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Close</button>
              <button type="button" class="btn btn-primary">Save changes</button>
            </div>
          </div>
        </div>
      </div>
