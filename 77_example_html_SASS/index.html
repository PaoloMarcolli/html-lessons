Tutorial Sass

Sass (Syntactically Awesome Style Sheets) è uno dei preprocessori CSS più popolari. Offre numerose funzionalità avanzate che migliorano la scrittura del CSS.

Trovi Less a questo link.

Le sue caratteristiche principali includono:

    Variabili: consentono di memorizzare valori CSS che possono essere riutilizzati. Vediamo un esempio. 
                                
scss

$primary-color: #3498db;
    body {
        color: $primary-color;
    }

  • Mixin: blocchi di codice riutilizzabili. Vediamo un esempio. 
                                
scss

    @mixin border-radius($radius) {
        -webkit-border-radius: $radius;
        -moz-border-radius: $radius;
        border-radius: $radius;
    }
    .box { 
        @include border-radius(10px); 
    }

  • Ereditarietà: consente di estendere le proprietà di un selettore ad altri selettori. Vediamo un esempio. 
                                
scss

    .message {
        border: 1px solid #ccc;
        padding: 10px;
        color: #333;
    }
    .success {
        @extend .message;
        border-color: green;
    }
