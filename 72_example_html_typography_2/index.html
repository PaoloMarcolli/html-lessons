Allineamento del testo

L'allineamento del testo ha un impatto su come leggiamo il contenuto. Probabilmente non ci hai mai fatto caso, ma i tuoi occhi non si muovono in modo fluido mentre leggono un paragrafo: saltano da una parola all'altra e da una riga all'altra, fissandosi su determinati punti e saltandone altri.

In un documento HTML ben progettato, l'allineamento del testo non è mai una decisione arbitraria. Un buon allineamento del testo facilita la lettura del contenuto, fornendo agli occhi un punto di riferimento su cui concentrarsi quando passano da una riga all'altra.

Le sezioni seguenti spiegano quando utilizzare l'allineamento a sinistra, al centro, a destra e giustificato. Tutti questi si basano sulla proprietà CSS text-align, che controlla l'allineamento del testo di un particolare elemento HTML.

Allineamento a sinistra

La maggior parte del testo dovrebbe essere allineata a sinistra, perché questo fornisce al lettore un ancoraggio verticale su cui tornare a ogni riga.

Le lunghe sezioni di testo, in particolare, dovrebbero quasi sempre essere allineate a sinistra. I testi brevi e i titoli offrono un po' più di flessibilità.

Ecco un esempio di codice CSS per allineamento a sinistra.

                    
                        p {
                            text-align: left;
                        }

Allineamento al centro

Il testo centrato non ha un ancoraggio, quindi è più facile per l'occhio perdersi quando cerca di saltare alla riga successiva.

È più adatto per righe di testo brevi e per contenuti speciali come poesie, testi di canzoni e titoli.

Ecco un esempio di codice CSS per allineamento al centro.

                    
                        .center {
                            text-align: center;
                          }

Allineamento a destra

Un altro aspetto da considerare nella scelta dell'allineamento del testo è il rapporto che crea con gli elementi circostanti. Per esempio, puoi allineare a destra una didascalia per farla sembrare associata a un'immagine.

Ecco un esempio di codice CSS per allineamento a destra.

                    
                        figcaption {
                            display: none;
                          }
                          @media only screen and (min-width: 900px) {
                            figure {
                              position: relative;
                            }
                            figcaption {
                              display: block;
                              font-style: italic;
                              text-align: right;
                              background-color: #FFFFFF;
                              position: absolute;
                              left: -220px;
                              width: 200px;
                            }
                          }

Testo giustificato

Il testo giustificato viene creato regolando sottilmente lo spazio tra parole/lettere e dividendo le parole lunghe con trattini, in modo che ogni riga abbia la stessa larghezza.

Senza un motore di sillabazione di alta qualità, il testo giustificato crea spazi irregolari tra le parole. Questi spazi irregolari rendono più difficile per l'occhio muoversi orizzontalmente attraverso il testo.

Ecco un esempio di codice CSS per allineamento giustificato.

                    
                        .justify {
                            text-align: justify;
                          }
